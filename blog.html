Topics Global Scope Block Scope
Position A variable declared outside a function, becomes GLOBAL Scope. Variables that declared inside a { } block are Block Scope.
Variable Declare Word var, let, const Variables declared with the var keyword can NOT have block scope.
Accessibility Global Scope variable can be accessed from anywhere. Variables declared inside a { } block cannot be accessed from outside the block.

Topics Local Storage Session Storage
Data Store Data is stored and is not cleared after closing the browser or tab. Data is stored temporary and cleard after closing the browser or tab.
Storage Capacity The storage capacity of local storage is 5MB/10MB. The storage capacity of session storage is 5MB.

The Event Loop is a browser mechanism which run JavaScript asynchronous programme. JavaScript is a single-thread programming language. So it cann't run a synchronous & asynchronous code at a time. But by the help of browser using Event Loop it can handle asynchronous behaviour. The Call Stack keep track of all the the function or code to be executed. When a function is finished, it is popped up from the stack. The Queue send new function to the stack. It's follow the data structure to maintain the correct sequence. Whenever an asynchronous function is called, it is sent to a browser API. These APIs built into the browser. Based on the command received from the call stack, the API start its own single-threaded operation. That is how Event Loop work.

We get undefined for various errors.
  • Variable that is not initialized will give undefined.
  • Function with no return will give undefined.
  • Parameter that is not passed as argument will be undefined.
  • If return value is nothing will return undefined.
  • Property that doesn't exist in an object will give undefined.
  • Accessing array element outside the index range will give undefined.
  • If we set a value Undefined will be undefined.
